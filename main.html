Question-1:What is the difference between innerHTML, innerText, and textContent ?

Answer: innerHTML এটি একটি element-এর সম্পূর্ণ HTML কোড (ট্যাগ সহ) প্রদান করে বা সেট করতে পারে। এ্বং innerText এটি শুধুমাত্র দৃশ্যমান (visible) টেক্সট প্রদান করে বা সেট করতে পারে। textContent এটি একটি element-এর সম্পূর্ণ টেক্সট (hidden বা visible) প্রদান করে বা সেট করতে পারে।

Question-3: What is event delegation in the DOM?

Answer: Event Delegation হল এমন একটি প্রযুক্তি যেখানে parent element-এর উপর event listener সেট করা হয়, যা তার child elements-এ সংঘটিত event গুলোকে হ্যান্ডেল করতে পারে। এটি event bubbling মেকানিজমের উপর ভিত্তি করে কাজ করে।
